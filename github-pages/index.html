Windows-first package manager

flget

Directory-isolated installs. Rename-based updates. Portable by default.

Run the update script from the directory where you want to create or refresh the flget root.

cd C:\tools\flget powershell -c "irm https://flatina.github.io/flget/update.ps1 -OutFile .\update.ps1" .\update.ps1
update.ps1 GitHub

The update script downloads flget.js, flget.js.map, bun.exe, activate.ps1, update.ps1, and REGISTER_PATH.ps1. The first activate run bootstraps flget.root.toml and creates shims/flget.* and shims/bun.*.
